# Restaurant Management System - Manager Manual

## Introduction
This system helps manage a restaurant's daily operations, including menu management, order taking, staff management, and sales analytics.

## Features

### 1. Menu Management
- **View Menu**: See all available items.
- **Add Item**: Add new dishes with price and category.
- **Delete Item**: Remove items from the menu.

### 2. Order Management
- **Take Order**: Create a new order for a table.
- **View Orders**: See all active and completed orders.
- **Complete Order**: Mark an order as paid/completed.

### 3. Staff Management
- **View Staff**: See list of employees.
- **Add Staff**: Register new staff members.

### 4. Analytics
- **Show Analytics**: View total revenue and a graph of daily sales.

## How to Run
1. Open a terminal.
2. Navigate to the project folder.
3. Run `python3 restaurant_management_system.py`.
